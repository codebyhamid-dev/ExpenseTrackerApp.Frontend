**ExpenseTrackerApp.Frontend**

A modern Expense Tracker frontend application built with Angular 17, designed to work seamlessly with the Expense Tracker ASP.NET Core Web API.
The application provides a clean, responsive, and user-friendly interface for managing income and expenses, with secure authentication and real-time data visualization.
